# Adolescent Height Analysis Across Racial/Ethnic Groups (YRBS 2007)

## 1. Research Question
* **Question**: Does adolescent height significantly differ across different racial or ethnic groups?
* **Objective**: To investigate whether race and ethnicity are significantly associated with variations in height among high school students using the YRBS 2007 dataset.

## 2. Variables Defined
* **Independent Variable (IV)**: `RaceEth` (Categorical variable representing 8 distinct racial/ethnic groups, coded 1 to 8).
* **Dependent Variable (DV)**: `HowTallAreYouWithoutShoesInMeters` (Continuous numerical variable representing height in meters).

## 3. Data Cleaning
* Utilized Python and `pandas` library to filter the raw dataset.
* Applied `.dropna()` to remove missing data (`NaN`) in both `RaceEth` and height columns to ensure the reliability and quality of the statistical analysis.

## 4. Statistical Method
* **Method**: **One-way ANOVA (Analysis of Variance)** was selected to compare the mean heights across the 8 racial/ethnic groups.
* **Post-hoc Test**: **Tukey's HSD (Honestly Significant Difference) Test** was used following a significant ANOVA result to conduct pairwise comparisons and identify which specific groups significantly differ from one another.

## 5. Key Results & Interpretation
* **ANOVA Results**: The One-way ANOVA yielded an **F-statistic of 36.1956** and a **p-value < 0.001** (highly significant). 
* **Hypothesis Conclusion**: Since $p < 0.05$, we strong **reject the null hypothesis** ($H_0$) and conclude that adolescent heights differ significantly across racial/ethnic groups.
* **Tukey's HSD Findings**:
  * **Group 3** has a significantly **higher** average height compared to Groups 1, 2, 4, 5, and 7.
  * **Group 5** has a significantly **lower** average height compared to Groups 1, 3, and 6.
